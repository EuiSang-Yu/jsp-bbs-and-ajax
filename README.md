### Description
* 라이엇의 리그오브레전드 게임의 전적 검색 및 공략을 공유할 수 있는 게시판이 있는 JSP기반 웹 사이트입니다.

### Purpose
* Ajax 를 이용해 라이엇의 공식 API키를 통해 JSON 데이터를 받아 파싱해서 화면에 보여주고, 원하는 게임 캐릭터의 공략을 공유할 수 있는 게시판 기능 구현
